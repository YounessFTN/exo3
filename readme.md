# Lancer le projet Python

## Sur Mac

1. **Installer Python (via Homebrew)**

```bash
brew install python
```

2. **Installer les modules nécessaires**

```bash
pip3 install pandas
pip3 install plotly
```

3. **Lancer le projet**

```bash
python3 index.py
```

---

## Sur Windows

1. **Installer Python**

- Télécharger et installer Python depuis [python.org](https://www.python.org/downloads/)
- **Important** : cocher “Add Python to PATH” pendant l’installation

2. **Installer les modules nécessaires**

```cmd
pip install pandas
pip install plotly
```

3. **Lancer le projet**

```cmd
python index.py
```
